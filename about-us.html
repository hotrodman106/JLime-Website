About Java Lime



About The Developers:

Java Lime is created by the evil masterminds Paul (@hotrodman106) and Jared (@Coolway99). In the beginning, Java Lime was developed as a way to parse commands for frequently used lines of Java code. However, as time went on, Paul and now fellow developer Jared realized that Java Lime had the potential to become a powerful language and soon it was further devolved creating the language we now love today.

About The Name:

The name Java Lime came from a statement made by the Paul, the original creator of the program, to his friend. When confronted about the similarities between his program and BASIC, he responded, "This program and BASIC are like lemons and limes, look similar but taste totally different". Shortly after adding the Java prefix, the name stuck.

About The Logo:

The Java Lime logo was created by Paul and it consists of two main parts. The first part being the lime which represents the statement as explained above. The second part being the Java cup which represents the fact that Java Lime is coded in Java.
